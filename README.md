# Анализ защищённости пакетов ALT Linux

Данный репозиторий содержит инструменты и результаты анализа защищённости исполняемых файлов в пакетах ALT Linux с использованием утилиты `checksec`.

## 📋 Содержание репозитория

- **`w.sh`** — основной исполняемый файл, который выполняет анализ защищённости. Скрипт обрабатывает вывод `checksec` и формирует отчёт в формате CSV.
- **`all.txt`** — накопленный результат работы `checksec` в формате CSV, содержащий данные обо всех проверенных файлах.
- **`log_all_bin.txt`** — подробный лог-файл с результатами проверки всех файлов и пакетов.

## 📁 Структура данных

Папки, **обозначенные ЗАГЛАВНЫМИ буквами**, содержат классифицированную информацию о наличии или отсутствии конкретных механизмов защиты в пакетах:

*Каждая такая директория соответствует определённому типу защиты и содержит списки пакетов, где эта защита включена или отключена.*

## 🔍 Пояснение вывода

Программа формирует структурированный вывод, который содержит следующую информацию:

1.  **Состояние защит** — для каждого проверенного файла отображается, какие механизмы защиты **включены**, а какие — **отключены**.
2.  **Информация о файле** — полный путь к проверяемому исполняемому файлу.
3.  **Принадлежность к пакету** — имя пакета, в состав которого входит данный файл.
4.  **Группа пакета** — группа, к которой относится пакет (например, `System/Base`, `Development/Libraries` и т.д.).

### ✨ Особенности

*   **Сортировка:** Все данные упорядочены по именам пакетов для удобства анализа.
*   **Отслеживание неоднородности:** Реализован механизм, который отслеживает **различия в конфигурации защит у разных файлов внутри одного пакета**. Это ключевая функция, позволяющая выявить файлы, чей профиль защиты выбивается из общей "кучи" пакета и требует отдельного внимания.
